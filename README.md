# DalamudMinesweeper

Dalamud plugin for FFXIV.

How to play:
- Left click to uncover a square, right click to place a flag.
- The game ends when all safe squares have been uncovered.
- Click the smiley face to start a new game.

Features:
- Score board
- No-guess mode!
- 4 difficulty presets
- Optional shortcuts
